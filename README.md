# Task-4-Power-Bi-Dashboard
# Financial Sales Dashboard - Power BI

## Project Overview
This project was created as part of a Data Analyst Internship (Task 4). The objective was to design an interactive Power BI dashboard that helps business stakeholders analyze sales performance, profit, products, countries, and business segments.

---

## Objective
- Build an interactive sales dashboard using Power BI.
- Track important business KPIs.
- Analyze sales trends over time.
- Provide insights for better business decision-making.

---

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Financial Sample Dataset

---

## Dataset
- Dataset Name: Financial Sample Dataset
- Source: Microsoft Sample Dataset

---

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Units Sold KPI
- Profit Margin KPI
- Monthly Sales Trend
- Sales by Country
- Sales by Product
- Profit by Product
- Sales by Segment
- Interactive Slicers (Year, Country, Product, Segment)

---

## Key Insights
- Sales performance can be monitored using KPI cards.
- Monthly sales trends help identify business growth.
- Country-wise and product-wise analysis supports business decision-making.
- Interactive slicers allow users to filter data dynamically.
